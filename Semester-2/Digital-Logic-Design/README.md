# Room Security and Automation 
**Course:** Digital Logic Design (Semester 2)

A comprehensive smart room system designed and simulated in **Multisim**. This project integrates binary password security, motion-based alarms, and automated appliance control.

##  Project Features
* **Access Control:** 5-bit binary digital lock with XNOR logic.
* **Security Alarm:** Automated siren triggered after 3 failed attempts (via JK Flip-Flops).
* **Automation:** Motion-sensing lights and temperature-regulated ventilation.
