# GemsCraft Solution 
**Course:** Object-Oriented Programming (Semester 2)

A comprehensive Java application for gemstone management, integrating Object-Oriented Programming and SQL database logic.

##  Project Highlights
* **Full System:** Includes Java source code and SQL database configurations.
* **OOP Implementation:** Uses Classes, Inheritance, and Exception Handling.
* **Database Driven:** Backend logic managed through SQL queries.
