# Int-Uni Multi-Campus Network Design 
**Course:** Computer Communication & Networks (Semester 3)
**Tool:** Cisco Packet Tracer

##  Project Overview
Design of a hierarchical network for a university with campuses in **Karachi, Lahore, and Islamabad**.

##  Key Features
* **Router-on-a-Stick (ROAS):** Efficient inter-VLAN routing using IEEE 802.1Q.
* **Segmentation:** VLANs for Students, SSC, and Marketing departments.
* **Security (ACLs):** Extended Access Control Lists to block specific traffic (e.g., blocking Facebook for students).
* **Star Topology:** Cost-effective and scalable campus design.
