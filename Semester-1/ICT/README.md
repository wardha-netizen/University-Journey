# ICT Final Project
**Course:** Introduction to Information & Communication Technology (Semester 1)

This project consists of a web-based component and full documentation regarding HealthCare Automation System.

##  Project Components

* **Project Documentation:** Complete set of requirements and technical docs (Found in the Docs ZIP).

* `ICT-Proposal.pdf` — The official project proposal.
