# Bank Management System 
**Course:** Computer Programming Lab (Semester 1)

A console-based C++ application designed to manage bank records efficiently.

##  Key Features
* **Account Management:** Create, Display, and Delete account records.
* **Transactions:** Secure Deposit and Withdrawal logic.
* **Data Persistence:** Records are saved to `accounts.txt`.
